Jarvis - description

  • A Personal Assistant for Linux, MacOS and Windows

  • Usage principal en ligne de commande

  • Fonctionnalités étendues par des pluggins

    • Pluggin pour ajouter des commandes
    • Pluggin pour que Jarvis parle
    • Pluggin pour parler à Jarvis
- Pas de DB : sauvegardé dans des fichier - Pas de GUI

Technologie

  • Python3
  • ... Et c'est tout :)
- data : endroit où sont sauvegardées les données - CmdInterpreter : contient JarvisAPI, Jarvis interface for plugins.

- data : endroit où sont sauvegardées les documents - packages/ memory : endroit où sont sauvegardées les données utilisateur - CmdInterpreter : contient JarvisAPI, Jarvis interface for plugins.
